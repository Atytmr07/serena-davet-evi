# Serena Organizasyon ve Davet Evi

Antalya Kepez'de bulunan 200 kişilik boutique davet salonunun web sitesi.
Modern Next.js 14, Tailwind CSS ve SEO odaklı mimari ile geliştirilmiştir.

## Özellikler

- **Modern Tasarım:** Glassmorphism, animasyonlar ve premium UI.
- **Galeri:** Next/Image optimizasyonu ve lightbox özelliği.
- **İletişim:** WhatsApp ve telefon entegrasyonu.
- **SEO & Performans:**
  - Dinamik Sitemap ve Robots.txt
  - JSON-LD Schema Markup (Event Venue)
  - Otomatik Meta Tag yönetimi
  - Mobil uyumlu (PWA Manifest)

## Kurulum

```bash
npm install
npm run dev
```

## Dağıtım (Deployment)

Bu proje Vercel üzerinde barındırılmaya hazırdır.

1.  GitHub'a Pushlayın.
2.  Vercel Dashboard'dan yeni proje oluşturun.
3.  GitHub reposunu seçin ve `Deploy` butonuna basın.
